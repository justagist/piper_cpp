# Piper CPP

- [piper_cpp](piper_cpp) -- C++ SDK for the AgileX Piper 6-DOF arm with gripper. This is a standalone pure C++ library.
- [piper_cpp_ros](piper_cpp_ros) -- A ros2_control hardware interface for the AgileX Piper 6-DOF arm, built on top of
`piper_cpp`.
- [piper_cpp_moveit](piper_cpp_moveit) -- MoveIt 2 configuration for the AgileX Piper 6-DOF arm.
