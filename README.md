# Piper CPP

- [piper_cpp](piper_cpp/README.md) -- C++ SDK for the AgileX Piper 6-DOF arm with gripper. See [README](piper_cpp/README.md).
- [piper_cpp_ros](piper_cpp_ros/README.md) -- A ros2_control hardware interface for the AgileX Piper 6-DOF arm, built on top of
